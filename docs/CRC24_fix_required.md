# Исправление CRC-24 в коде - ВЫПОЛНЕНО

## Проблема
В коде использовался неверный полином CRC-24 для Galileo. Старая реализация использовала полином 0x1864CFB, что соответствует устаревшей версии спецификации.

## Исправленные файлы:
1. `/src/NavBit.cpp` - таблица Crc24q[] исправлена
2. `/src/BCNavBit.cpp` - таблица crc24q[] исправлена

## Правильный полином:
Согласно актуальной спецификации Galileo OS SIS ICD:
- Полином: x^24 + x^23 + x^18 + x^17 + x^14 + x^11 + x^10 + x^7 + x^6 + x^5 + x^4 + x^3 + x + 1
- Hex: 0x864CFB

## Результат:
Таблицы CRC-24 пересчитаны с правильным полиномом. Теперь страницы F/NAV будут проходить проверку CRC в приёмниках.
